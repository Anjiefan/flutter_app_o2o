# O2O - Flutter 版校园o2o食堂预定app

- [English](https://github.com/Mayandev/morec/blob/master/README_EN.md)

这是一款完整的、商业级开源应用，旨在解决全国高校校内餐饮、零售等服务的效率问题，实现数字化校园，智能化校园。
已实现的功能有：
- [x] 线上下单
- [x] 快速预定
- [x] 货币系统
- [x] 分销系统
- [x] 优惠卷系统
- [x] 店铺管理
- [x] 商品管理
- [x] 订单管理

未来可能升级的功能有：
- 硬件系统
- 跑腿系统
- 校外餐饮优惠卷系统
